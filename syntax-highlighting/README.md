# QTMX- Syntax Highlighting

Complete syntax highlighting for QTMX- Pure language.

## Installation

### VS Code
1. Copy all files to `~/.vscode/extensions/qtmx-syntax-1.0.0/`
2. Restart VS Code
3. Select QTMX- language for .qtmx- files

### Sublime Text
1. Copy qtmx.json to `~/.config/sublime-text-3/Packages/QTMX/`
2. Restart Sublime

### Atom
1. Copy qtmx.json to `~/.atom/packages/qtmx-syntax/`
2. Restart Atom

## Features

✅ Syntax highlighting for beQ-, qq-, bbq, aaq, ingo, outgo, tcq
✅ Color themes (dark)
✅ Auto-indentation
✅ Bracket matching
✅ Comment support

## Colors

- Keywords (beQ-, qq-): Blue
- Blocks (bbq, aaq): Cyan
- I/O (ingo, outgo): Light Blue
- Validation (tcq, check): Yellow
- Declarations: Orange
- Types: Green
- Strings: Orange
- Numbers: Light Green
- Comments: Gray-Green
